loop

循环语句，通常与 exitwhen 配合控制退出条件。

语法

loop
    exitwhen condition
    // body
endloop

示例

set i = 0
loop
    exitwhen i >= 10
    set i = i + 1
endloop

loop 内提前退出：exitwhen

exitwhen 是退出当前循环的标准方式，推荐优先使用。

set i = 0
loop
    exitwhen i >= 10
    set i = i + 1
endloop

loop 内提前退出：return

return 会直接结束整个函数，不只是跳出循环。

function FindFirstAlive takes unit array arr, integer n returns integer
    local integer i = 0
    loop
        exitwhen i >= n
        if arr[i] != null and GetWidgetLife(arr[i]) > 0.405 then
            return i // 直接结束函数
        endif
        set i = i + 1
    endloop
    return -1
endfunction

怎么选？

  • 只想结束循环，后面还有逻辑要执行：用 exitwhen。
  • 找到结果后要立刻结束整个函数：用 return。
