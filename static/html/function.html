function

定义函数，是 JASS 最核心的组织单元。函数可包含参数、返回值、局部变量与流程控制。

语法模板

function name takes params returns returnType
    // body
endfunction

完整示例

function Add takes integer a, integer b returns integer
    if a == 0 and b == 0 then
        return 0
    endif
    return a + b
endfunction

建议

  • 参数和返回值写清晰，避免“万能函数”。
  • 异常条件建议前置并提前返回。
  • 保证所有返回路径显式 return，避免历史兼容歧义。
