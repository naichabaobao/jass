library

语法

library 名称 [initializer InitFunc] [requires|needs|uses [optional] 依赖, ...]
    // 内容
endlibrary

说明

定义库并参与编译期排序。文档说明库可声明依赖，JassHelper 会按依赖调整顺序，并可配合 initializer 在初始化阶段执行。

示例

library Core initializer InitCore requires optional Utils
function InitCore takes nothing returns nothing
endfunction
endlibrary

注意事项

  • 库名区分大小写；禁止循环依赖；requires/needs/uses 在文档中语义等价。
