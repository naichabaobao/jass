initializer

语法

library L initializer InitL
function InitL takes nothing returns nothing
endfunction

说明

库初始化入口，按依赖顺序执行。文档强调 initializer 函数不能有参数。

示例

library A initializer InitA requires B
function InitA takes nothing returns nothing
endfunction
endlibrary

注意事项

  • 当 A 依赖 B 时，B 的 initializer 先于 A 执行。
